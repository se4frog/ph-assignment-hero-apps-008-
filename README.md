# App Store Clone

A responsive **App Store clone** built with React, featuring live search, app installation via **localStorage**, and dynamic charts.

## Features

- **Home & Navigation:** Banner, top apps, and active route links
- **All Apps Page:** Search, filters, and app listing
- **App Details:** Install apps, show reviews chart, and descriptions
- **My Installation:** View installed apps, uninstall, and sort by downloads
- **Extras:** Loading animations, custom footer, and error page

## Tech Stack

React | React Router | Recharts | LocalStorage | Tailwind CSS
